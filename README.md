# FERNA-BETTY-DATABASE-PROJECT-
Billing and Payments management system is a relational database system designed to manage the complete financial life cycle of a hotel guest, from room booking through invoice generation to payment processing and receipt issuance. The system supports hotel front desk staff, cashiers, and managers in efficiently handling billing operations, reducing manual errors, and maintaining accurate financial records.
The Hotel Billing and Payment System supports     
  Recording guest information and room bookings
  Generating invoices that consolidate room charges and additional services
  Processing payments from guests using various payment methods such as cash, card, and mobile money
  Issuing receipts as proof of payment
  Tracking outstanding balances and payment status
